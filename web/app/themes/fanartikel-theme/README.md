# Fanartikel Theme

Modern, responsive WordPress theme for Deine Fanartikel.

## Features

- Modern, mobile-first design
- Responsive layout
- Custom homepage with sections:
  - Hero section with CTA
  - Product showcase
  - Konfigurator teaser
  - Advantages/USP section
  - Contact form integration
- Smooth animations
- SEO optimized
- Fast loading

## Installation

This theme is part of the Bedrock WordPress setup. It will be automatically available after deployment.

## Activation

1. Log into WordPress admin
2. Go to Appearance > Themes
3. Activate "Fanartikel Theme"

## Customization

- Logo: Upload via Appearance > Customize > Site Identity
- Menus: Configure via Appearance > Menus
- Contact Form: Install and configure Contact Form 7
